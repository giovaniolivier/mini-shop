# Mini-Shop

Un projet e-commerce simple avec une architecture microservices.

## Structure du projet

- `frontend/` - Interface utilisateur React
- `product-service/` - Service de gestion des produits
- `create-admin.js` - Script pour créer un administrateur

## Installation

### Prérequis
- Node.js (version 14 ou supérieure)
- npm ou yarn

### Installation des dépendances

1. Installer les dépendances du frontend :
```bash
cd frontend
npm install
```

2. Installer les dépendances du service de produits :
```bash
cd product-service
npm install
```

## Démarrage

### Frontend
```bash
cd frontend
npm start
```

### Service de produits
```bash
cd product-service
npm start
```

## Fonctionnalités

- Gestion des produits
- Interface utilisateur moderne
- Architecture microservices
- API RESTful

## Technologies utilisées

- **Frontend** : React
- **Backend** : Node.js, Express
- **Base de données** : MongoDB
